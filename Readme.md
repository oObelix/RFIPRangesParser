## Russian Federation IP Address Ranges Parser

Parser for page: (https://lite.ip2location.com/russian-federation-ip-address-ranges)

### Requirements
- Python 3.7.5+ 
- Tornado 6.0.4+
- SQLAlchemy 1.3.20+ 
- PostgreSQL 11+

### Install
- pip install -r requirements.txt
- python run_scraper.py

### Run server
- python run_api.py
